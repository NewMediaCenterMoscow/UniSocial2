UniSocial
=========
UniSocial - system for data collection from social networks, such as VKontakte (VK), Facebook and Twitter.

Current status
--------------
System is in active development!

### VKontakte

Main methods:
 * users.get
 * friends.get
 * users.getSubscriptions
 * groups.getById
 * groups.getMembers
 * wall.get
 * wall.getReposts
